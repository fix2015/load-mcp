Getting Started

Overview

load-mcp aggregates MCP servers from multiple GitHub sources into a searchable registry. Install any servers with one command — no cloning repos or copying files.

Quick Start

# Install a servers
npx load-mcp install <name>

# Install for Cursor
npx load-mcp install <name> --tool cursor

# Install globally
npx load-mcp install <name> --global

# Search
npx load-mcp search react

# List all
npx load-mcp list

Commands

Install

load-mcp install <name>                    # Install for Claude Code (default)
load-mcp install <name> --tool cursor       # Install for Cursor
load-mcp install <name> --tool codex        # Install for Codex
load-mcp install <name> --global             # Install globally
load-mcp install <name> -o ./path           # Custom output path

Search

load-mcp search react                       # Search by keyword
load-mcp search "api design" --tag backend  # Search with tag filter
load-mcp search react --json                # JSON output

List

load-mcp list                                # List all
load-mcp list --tag testing                  # Filter by tag
load-mcp list --tool cursor                  # Filter by tool
load-mcp list --json                         # JSON output

Info

load-mcp info <name>                         # Detailed info
load-mcp info <name> --json                 # JSON output

Tags & Sources

load-mcp tags                                # Show all tags with counts
load-mcp sources                             # Show all sources

Sources

Skills are aggregated from multiple GitHub repositories maintained by the community.

Update Registry

load-mcp update                              # Fetch latest from GitHub
GITHUB_TOKEN=ghp_xxx load-mcp update         # With auth for higher rate limits

JSON Output

load-mcp list --json
load-mcp search react --json
load-mcp info <name> --json

All commands support --json for machine-readable output.

Programmatic API

const { findSkill, searchSkills, installSkill } = require('load-mcp');

const results = searchSkills('react', { tag: 'frontend' });
const skill = findSkill('react-expert');
await installSkill(skill, { tool: 'claude-code', global: true });
